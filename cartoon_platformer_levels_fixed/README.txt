Запуск игры без чёрного экрана (важно):
1) Откройте папку проекта в VS Code.
2) Установите расширение "Live Server" (если ещё не установлено).
3) Кликните правой кнопкой по index.html → "Open with Live Server".
   Адрес будет вида http://127.0.0.1:5500/
   Почему так? Phaser грузит ассеты через XHR, а при открытии файла напрямую (file://) браузер блокирует запросы (CORS).

Альтернативы Live Server:
- Python 3:  в терминале (на уровне index.html) выполните
    python -m http.server 8000
  затем откройте http://localhost:8000
- Node.js: 
    npx http-server
  затем откройте указанный адрес.

Если всё равно чёрный экран:
- Откройте DevTools (F12) → вкладка Console и посмотрите ошибки.
- Убедитесь, что есть интернет (CDN Phaser).
- Сообщите мне текст ошибок, я помогу исправить.
